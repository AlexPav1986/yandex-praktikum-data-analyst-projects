# Дашборт для Яндекс.Дзен

## Задача:
Используя данные Яндекс.Дзена построить дашборд с метриками взаимодействия пользователей с карточками статей 

## Основные дейстивия:
- дашборд
- пайплайн
- Yandex.Cloud
- удаленный сервер
- виртуальная машина

## Используемые иснтрументы:
- Python
- SQLAlchemy
- PostgreSQL
- Tableau
- продуктовые метрики
- построение дашбордов
